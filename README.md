Hospital Emergency Room Analytics – Excel Dashboard-

Project Overview-
This project analyzes hospital Emergency Room (ER) operations using Excel, Power Query, Power Pivot, DAX, and interactive dashboards.
The goal is to help hospital management improve patient flow, reduce wait times, and understand demographic and departmental trends.

Business Requirement Document (BRD) KPIs-
⦁	Total Number of Patients

⦁	Average Wait Time (Minutes)

⦁	Patient Satisfaction Score

⦁	% of Patients Seen Within 30 Minutes

⦁	Admission Status Breakdown

Charts-
⦁	Patient Admission Status

⦁	Patient Age Distribution

⦁	Timeliness (On-time vs Delayed)

⦁	Gender Analysis

⦁	Department Referral Analysis

⦁	Monthly & Daily Patient Trend

Project Steps-
⦁	Business Requirement Gathering

⦁	Understanding the Data

⦁	Import & Clean Data using Power Query

⦁	Create Calendar Table (Power Query)

⦁	Data Modeling in Power Pivot (Relationships)

⦁	Create DAX Measures (Wait Time, Satisfaction, Seen-on-Time, etc.)

⦁	Build PivotTables and Charts

⦁	Dashboard Layout & Formatting

⦁	Insights Generation

Tools & Techniques Used-
⦁	Microsoft Excel

⦁	Power Query (ETL)

⦁	Power Pivot (Data Model)

⦁	DAX (Measures)

⦁	PivotTables

⦁	PivotCharts

⦁	Slicers

⦁	Conditional Formatting

Key Insights-
⦁	ER handled 513 patients in the January month.

⦁	36.32 minutes average wait time — improvement needed.

⦁	62% of patients attended within 30 minutes; 38% delayed.

⦁	Patient satisfaction score: 4.96

⦁	Age groups 10–19 and 20–29 have the highest ER visits.

⦁	Gender split is nearly balanced.

⦁	General Practice receives highest department referrals.
